## Python datetime フォーマット一覧

Python の `datetime.strftime()` および `datetime.strptime()` で使用できる  
日付・時刻フォーマット指定子の一覧です。

### 日付に関するフォーマット

| フォーマット | 意味 | 例 |
|--------------|------|------|
| `%Y` | 西暦（4桁） | 2026 |
| `%y` | 西暦（下2桁） | 26 |
| `%m` | 月（01–12） | 02 |
| `%B` | 月名（英語・フル） | February |
| `%b` / `%h` | 月名（英語・短縮） | Feb |
| `%d` | 日（01–31） | 18 |
| `%j` | 年内通算日（001–366） | 049 |
| `%U` | 年内週番号（週の開始：日曜） | 07 |
| `%W` | 年内週番号（週の開始：月曜） | 07 |

### 時刻に関するフォーマット

| フォーマット | 意味 | 例 |
|--------------|------|------|
| `%H` | 時（00–23） | 10 |
| `%I` | 時（01–12） | 10 |
| `%p` | AM/PM | AM |
| `%M` | 分（00–59） | 00 |
| `%S` | 秒（00–59） | 30 |
| `%f` | マイクロ秒（000000–999999） | 123456 |
| `%z` | UTC オフセット | +0900 |
| `%Z` | タイムゾーン名 | JST |

### 日付＋時刻の複合

| フォーマット | 意味 | 例 |
|--------------|------|------|
| `%c` | ロケール依存の日時表現 | Tue Feb 18 10:00:00 2026 |
| `%x` | ロケール依存の日付表現 | 02/18/26 |
| `%X` | ロケール依存の時刻表現 | 10:00:00 |

### その他

| フォーマット | 意味 | 例 |
|--------------|------|------|
| `%%` | `%` 自身を出力 | % |

print(datetime.datetime.strptime(dt_srt,"%Y-%m-%dT%H:%M:%S.%f%z"))
