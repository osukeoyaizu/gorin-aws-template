## Application Recovery Controller
### 前提条件
ap-northeast-1,us-east1両リージョンにALB,AutoScaling,DynamoDBが存在する(同じ構成)

#### リカバリグループ作成
セルをリージョン毎に作成

#### リソースセット作成
ALB / AutoScaling Group / DynamoDB毎に作成

#### 準備状況チェック作成
リソースセット毎に作成

適切なセルを選択

#### クラスター作成

#### コントロールパネル作成

#### ルーティングコントロール作成
ALB毎に作成

#### 安全ルール作成
ルールタイプ：アサーションルール

コントロール：ルーティングコントロールを選択

ルール：少なくとも1


#### ヘルスチェック作成
ルーティングコントロール毎に作成

#### レコードセット作成
Route53から作成されたヘルスチェックを使用し、ルーティングポリシー(フェイルオーバー)でALBエイリアスレコードを作成する

#### 確認
ルーティングコントロールの状態をオンにする　→　ヘルスチェックが「Healthy」になる
