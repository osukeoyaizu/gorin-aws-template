## Lake Formationで管理する
### Lake Formationの管理者を設定
「Lake Formation」→「Administration」→「Administrative roles and tasks」で管理者を選択する

### Data locationsの設定
「Lake Formation」→「Permissions」→「Data locations」でデータの保存場所(S3パス)を登録する

### IAM, Lake Formationの両方でアクセス制限をする
「Lake Formation」→「Permissions」→「Hybrid access mode」でデータカタログに対してIAMロールを選択する
